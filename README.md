# Allison Batoff’s Portfolio (v2) - Front-end

## Description

Second version of UX Designer and Photographer Allison Batoff’s portfolio website. This is the front-end, built with [React](https://react.dev/) and [Tailwind CSS](https://tailwindcss.com/). For the back-end, see [this repository](https://github.com/JSRobertOuimet/allison-batoff-portfolio-v2-backend) (private).
